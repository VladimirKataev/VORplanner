# Victor Airway Planner

## Pathfinding along victor airways
- Ceiling support (FL125 or otherwise)
- Path Length support
- [Publically Available data](https://ais-faa.opendata.arcgis.com/datasets/acf64966af5f48a1a40fdbcb31238ba7_0/explore)
- Browser support

## Important note:
This is a hobby project, there is no warranty, expectation of accuracy or upkeep with this software. Please validate all results personally, every time, before using them.
